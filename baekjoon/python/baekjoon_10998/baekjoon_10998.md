# 풀이 과정
# 백준 10998번: A×B (Python)

## 문제 설명

두 정수 `A`와 `B`가 주어졌을 때, 두 수의 곱을 출력하는 문제이다.
입력은 한 줄로 주어지며, 공백을 기준으로 두 정수가 구분된다.

---

## 입력 조건

* 입력: 한 줄에 정수 `A`, `B`
* 범위: 문제에서 주어지는 일반적인 정수 범위

예시 입력

```
3 4
```

---

## 출력 조건

* `A × B`의 결과를 출력한다.

예시 출력

```
12
```

---

## 접근 방법

1. `input()`을 통해 한 줄을 입력받는다.
2. 공백을 기준으로 문자열을 분리하기 위해 `split()`을 사용한다.
3. 입력값은 문자열이므로 `int()`로 형 변환을 수행한다.
4. 두 정수를 곱한 결과를 출력한다.
5. 실행 진입점을 명확히 하기 위해 `if __name__ == '__main__':` 구문을 사용한다.

이 문제는 연산 자체가 단순하므로, 입력 처리와 형 변환이 핵심이다.

---

## 코드 구현

```python
def print_A_mul_B():
    A, B = input().split()
    print(int(A) * int(B))

if __name__ == '__main__':
    print_A_mul_B()
```

---

## 코드 설명

* `print_A_mul_B()`

  * 입력을 받아 두 값을 분리한 뒤 곱셈 결과를 출력하는 함수
* `input().split()`

  * 공백 기준으로 문자열을 나누어 `A`, `B`에 할당
* `int(A) * int(B)`

  * 문자열을 정수로 변환 후 곱셈 수행
* `if __name__ == '__main__':`

  * 해당 파일이 직접 실행될 때만 함수가 호출되도록 설정

---

## 시간 및 공간 복잡도

* 시간 복잡도: `O(1)`
* 공간 복잡도: `O(1)`

입력 크기와 무관하게 상수 시간 내에 해결된다.

기본적인 입력 처리와 정수 변환을 연습하기에 적합한 문제이다.
