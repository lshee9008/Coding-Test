## 문제 설명

불기 연도는 서기 연도보다 543년 빠르다.
불기 연도를 입력받았을 때, 이를 서기 연도로 변환하여 출력하는 문제이다.

---

## 입력

* 불기 연도 `A`가 주어진다.
* `A`는 정수이다.

## 출력

* 입력으로 주어진 불기 연도를 서기 연도로 변환한 값을 출력한다.

---

## 풀이 접근

불기(BE)와 서기(AD)의 차이는 543년이다.
즉, 불기 연도에서 543을 빼면 서기 연도가 된다.

문제 해결 흐름은 다음과 같다.

1. `input()`으로 불기 연도를 입력받는다.
2. 정수형으로 변환한다.
3. 불기 연도에서 543을 빼서 서기 연도를 계산한다.
4. 계산된 결과를 출력한다.

---

## 코드

```python
def change_yesrs():
    A = int(input())
    # 544년도를 빼야함 그리고 현재 년도 +1
    print(A - 544 + 1)

if __name__ == '__main__':
    change_yesrs()
```

---

## 코드 분석

### 입력 처리

```python
A = int(input())
```

* `input()`으로 값을 입력받는다.
* 연도 계산을 위해 정수형으로 변환한다.

---

### 연도 변환 계산

```python
print(A - 544 + 1)
```

* 불기 연도에서 544를 빼고 1을 더한 연산은
  결과적으로 불기 연도에서 543을 빼는 것과 같다.
* 이는 불기 연도와 서기 연도의 차이를 반영한 계산이다.

---

### 메인 실행부

```python
if __name__ == '__main__':
    change_yesrs()
```

* 해당 파일이 직접 실행될 때만 함수가 호출된다.
* 다른 파일에서 import 되었을 때 자동 실행되는 것을 방지한다.

* 입력 처리와 간단한 산술 연산 연습에 적합하다.
