# 문제 출처
> https://www.acmicpc.net/problem/1008

# 풀이 과정
## 문제 설명

두 정수 `A`와 `B`가 주어졌을 때,
`A ÷ B`의 결과를 출력하는 문제이다.

---

## 입력

* 한 줄에 정수 `A`, `B`가 공백으로 구분되어 주어진다.

## 출력

* `A / B`의 결과를 출력한다.
* 파이썬에서는 `/` 연산자를 사용할 경우 실수형 결과가 출력된다.

---

## 풀이 접근

입력값은 문자열 형태로 들어오기 때문에
나눗셈 연산을 위해 정수형으로 변환이 필요하다.

문제 해결 흐름은 다음과 같다.

1. `input()`으로 한 줄 입력을 받는다.
2. `split()`으로 공백 기준으로 값을 분리한다.
3. 각 값을 `int()`로 변환한다.
4. `/` 연산자를 사용해 나눗셈을 수행한다.
5. 결과를 출력한다.

연산 로직은 함수로 분리하고,
`__main__` 구문을 사용해 실행 흐름을 명확히 했다.

---

## 코드

```python
def print_A_div_B():
    A, B = input().split()
    print(int(A) / int(B))

if __name__ == '__main__':
    print_A_div_B()
```

---

## 코드 분석

### 함수 정의

```python
def print_A_div_B():
```

* 두 수를 입력받아 나눗셈 결과를 출력하는 함수이다.
* 기능 단위로 분리하여 코드의 역할을 명확히 했다.

---

### 입력 처리

```python
A, B = input().split()
```

* `input()`은 한 줄을 문자열로 입력받는다.
* `split()`을 사용해 공백 기준으로 두 값을 나눈다.
* 이 시점의 `A`, `B`는 문자열이다.

---

### 형 변환 및 나눗셈

```python
print(int(A) / int(B))
```

* 문자열 상태로는 산술 연산이 불가능하므로 `int()`로 변환한다.
* `/` 연산자는 정수끼리 연산하더라도 실수형 결과를 반환한다.
* 문제 조건에 맞게 별도의 출력 형식 지정 없이 그대로 출력한다.

---

### 메인 실행부

```python
if __name__ == '__main__':
    print_A_div_B()
```

* 해당 파일이 직접 실행될 때만 함수가 호출되도록 설정했다.
* 다른 파일에서 import 시 실행되는 것을 방지한다.

---

## 정리

* `/` 연산자는 항상 실수형 결과를 반환한다는 점을 기억해야 한다.
* 입력 → 분리 → 형 변환 → 연산 → 출력의 기본 흐름을 연습하기 좋은 문제이다.
* 이후 실수 출력 포맷이나 정밀도를 요구하는 문제의 기초가 된다.
