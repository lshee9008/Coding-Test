# 풀이 과정
# [백준] 10869번 사칙연산 (Python)
## 문제 설명

두 정수 `A`와 `B`가 주어졌을 때,
다음 다섯 가지 연산 결과를 순서대로 출력하는 문제이다.

1. 덧셈
2. 뺄셈
3. 곱셈
4. 몫
5. 나머지

---

## 입력

* 한 줄에 두 정수 `A`, `B`가 공백으로 구분되어 주어진다.

## 출력

아래 연산 결과를 각각 한 줄에 하나씩 출력한다.

* `A + B`
* `A - B`
* `A * B`
* `A // B`
* `A % B`

---

## 풀이 접근

입력값은 문자열 형태로 들어오기 때문에
산술 연산을 수행하기 위해 정수형으로 변환이 필요하다.

이 문제에서는 다음과 같은 방식으로 해결하였다.

1. `input().split()`으로 입력을 공백 기준으로 나눈다.
2. `map(int, ...)`을 사용해 입력과 동시에 정수형으로 변환한다.
3. 리스트로 저장해 인덱스로 각 값을 사용한다.
4. 사칙연산 결과를 순서대로 출력한다.
5. 실행 흐름을 명확히 하기 위해 `__main__` 구문을 사용한다.

---

## 코드

```python
def calculation():
    lst = list(map(int, input().split()))

    print(lst[0] + lst[1])
    print(lst[0] - lst[1])
    print(lst[0] * lst[1])
    print(lst[0] // lst[1])
    print(lst[0] % lst[1])

if __name__ == '__main__':
    calculation()
```

---

## 코드 분석

### 입력 처리 및 형 변환

```python
lst = list(map(int, input().split()))
```

* `input()`으로 한 줄을 입력받는다.
* `split()`으로 공백 기준 문자열을 분리한다.
* `map(int, ...)`을 사용해 각 값을 정수형으로 변환한다.
* 변환된 정수들을 리스트로 저장한다.

리스트의

* `lst[0]` → 첫 번째 정수
* `lst[1]` → 두 번째 정수
  를 의미한다.

---

### 사칙연산 출력

```python
print(lst[0] + lst[1])
print(lst[0] - lst[1])
print(lst[0] * lst[1])
print(lst[0] // lst[1])
print(lst[0] % lst[1])
```

* 덧셈(`+`), 뺄셈(`-`), 곱셈(`*`)은 기본적인 산술 연산이다.
* `//`는 나눗셈의 몫을 구하는 연산자이다.
* `%`는 나머지를 구하는 연산자이다.
* 각 연산 결과를 문제에서 요구한 순서대로 출력한다.

---

### 메인 실행부

```python
if __name__ == '__main__':
    calculation()
```

* 해당 파일이 직접 실행될 경우에만 함수가 호출된다.
* 다른 파일에서 import 되었을 때 자동 실행되는 것을 방지한다.

---

## 정리

* `map(int, input().split())`은 입력과 형 변환을 동시에 처리할 수 있어 자주 사용된다.
* 리스트를 사용하면 인덱스로 값에 접근할 수 있어 코드가 간결해진다.
* 사칙연산 문제는 파이썬의 기본 연산자 동작을 익히기에 적합하다.
