# [백준] 2588번 곱셈 (Python)

문제 사이트
[https://www.acmicpc.net/problem/2588](https://www.acmicpc.net/problem/2588)

---

## 문제 설명

(세 자리 수) × (세 자리 수)의 곱셈 과정을 출력하는 문제이다.
두 개의 자연수 `A`, `B`가 주어졌을 때,
아래와 같은 형식으로 계산 결과를 출력해야 한다.

1. `A`와 `B`의 일의 자리 곱
2. `A`와 `B`의 십의 자리 곱
3. `A`와 `B`의 백의 자리 곱
4. `A × B`의 최종 결과

---

## 입력

* 첫째 줄에 자연수 `A`
* 둘째 줄에 자연수 `B`
* 두 수는 모두 세 자리 수이다.

## 출력

* 문제에서 요구한 네 줄의 계산 결과를 순서대로 출력한다.

---

## 풀이 접근

이 문제의 핵심은 **정수에서 각 자릿수를 분리하는 방법**이다.

풀이 흐름은 다음과 같다.

1. 두 수를 문자열로 입력받는다.
2. 정수형으로 변환해 계산이 가능하도록 한다.
3. `B`의 각 자릿수(백의 자리, 십의 자리, 일의 자리)를 분리한다.
4. `A`에 각 자릿수를 곱한 결과를 출력한다.
5. 마지막으로 `A × B`의 최종 결과를 출력한다.

---

## 코드

```python
def multiple():
    A = input()
    B = input()
    A = int(A)
    B = int(B)
    B_list = [(B // 100), (B // 10) % 10, (B % 10)]
    print(A * B_list[2])
    print(A * B_list[1])
    print(A * B_list[0])
    print(A * B)

if __name__ == "__main__":
    multiple()
```

---

## 코드 분석

### 입력 처리 및 형 변환

```python
A = input()
B = input()
A = int(A)
B = int(B)
```

* 두 줄에 걸쳐 입력을 받는다.
* 계산을 위해 문자열을 정수형으로 변환한다.

---

### 자릿수 분리

```python
B_list = [(B // 100), (B // 10) % 10, (B % 10)]
```

* `B // 100` → 백의 자리
* `(B // 10) % 10` → 십의 자리
* `B % 10` → 일의 자리
* 각 자릿수를 리스트에 저장해 인덱스로 접근한다.

---

### 부분 곱 출력

```python
print(A * B_list[2])
print(A * B_list[1])
print(A * B_list[0])
```

* 문제에서 요구한 출력 순서에 맞게
  일의 자리 → 십의 자리 → 백의 자리 순으로 곱한 값을 출력한다.

---

### 최종 곱 출력

```python
print(A * B)
```

* 두 수의 전체 곱셈 결과를 출력한다.

---

## 정리

* 정수의 각 자릿수를 분리하는 방법은 자주 사용되는 패턴이다.
* 나눗셈(`//`)과 나머지(`%`) 연산을 함께 사용하면 쉽게 처리할 수 있다.
* 문제 요구사항에 맞는 출력 순서를 정확히 지키는 것이 중요하다.
