# [백준] 10171번 고양이 (Python)

문제 사이트
[https://www.acmicpc.net/problem/10171](https://www.acmicpc.net/problem/10171)

---

## 문제 설명

아래와 같은 고양이 모양의 ASCII 아트를 출력하는 문제이다.

```
\    /\
 )  ( ')
(  /  )
 \(__)|
```

출력 형식이 정확히 일치해야 하며,
공백과 특수문자도 모두 포함하여 출력해야 한다.

---

## 입력

* 입력은 없다.

## 출력

* 문제에서 제시한 고양이 모양을 그대로 출력한다.

---

## 풀이 접근

이 문제는 입력이나 연산이 없고,
**문자열을 정확히 출력하는지**를 확인하는 문제이다.

파이썬에서는 다음 두 가지를 주의해야 한다.

1. `\`는 이스케이프 문자이므로 문자열 안에서 `\\`로 작성해야 한다.
2. 공백과 줄바꿈 위치가 출력 결과에 그대로 반영되므로
   출력 문자열을 문제와 동일하게 맞춰야 한다.

---

## 코드

```python
def cat():
    print("\\    /\\")
    print(" )  ( ')")
    print("(  /  )")
    print(" \\(__)|")

if __name__ == "__main__":
    cat()
```

---

## 코드 분석

### 이스케이프 문자 처리

```python
print("\\    /\\")
```

* 문자열에서 `\`는 특수 문자이므로 `\\`로 작성해야 실제 `\`가 출력된다.
* `/\`를 출력하기 위해 `/\\` 형태로 작성되었다.

---

### 문자열 출력

```python
print(" )  ( ')")
print("(  /  )")
print(" \\(__)|")
```

* 각 줄을 `print()`로 한 줄씩 출력한다.
* 공백의 개수와 문자 위치가 결과에 그대로 영향을 주기 때문에
  문제에서 제시한 형식과 정확히 맞춰야 한다.

---

### 메인 실행부

```python
if __name__ == "__main__":
    cat()
```

* 해당 파일이 직접 실행될 때만 함수가 호출된다.
* 다른 파일에서 import 되었을 경우 자동 실행을 방지한다.

---

## 정리

* 출력 문제에서는 연산보다 **출력 형식의 정확성**이 가장 중요하다.
* 파이썬에서 `\`를 출력하려면 반드시 이스케이프 처리(`\\`)가 필요하다.
* ASCII 아트 문제는 문자열 처리와 출력 연습에 적합하다.
