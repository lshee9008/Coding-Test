# [백준] 11382번 꼬마 정민 (Python)

# 문제 사이트
> [https://www.acmicpc.net/problem/11382](https://www.acmicpc.net/problem/11382)

---

## 문제 설명

꼬마 정민이는 세 개의 정수 `A`, `B`, `C`를 받는다.
이 세 정수의 합을 출력하는 프로그램을 작성하는 문제이다.

---

## 입력

* 한 줄에 세 정수 `A`, `B`, `C`가 공백으로 구분되어 주어진다.
* 각 정수는 매우 큰 값일 수 있다.

## 출력

* `A + B + C`의 결과를 출력한다.

---

## 풀이 접근

이 문제는 여러 개의 정수를 한 줄에서 입력받아
모두 더하는 기본적인 입출력 문제이다.

파이썬에서는 다음과 같은 방식으로 간단히 해결할 수 있다.

1. `input().split()`으로 입력값을 공백 기준으로 분리한다.
2. `map(int, ...)`을 사용해 문자열을 정수형으로 변환한다.
3. 세 정수를 더한 결과를 출력한다.

---

## 코드

```python
def kid_jungmin():
    A, B, C = map(int, input().split())
    print(A + B + C)

if __name__ == '__main__':
    kid_jungmin()
```

---

## 코드 분석

### 입력 처리 및 형 변환

```python
A, B, C = map(int, input().split())
```

* `input()`으로 한 줄을 입력받는다.
* `split()`으로 공백을 기준으로 문자열을 나눈다.
* `map(int, ...)`을 사용해 세 값을 모두 정수형으로 변환한다.
* 변환된 값이 각각 `A`, `B`, `C`에 할당된다.

---

### 합 계산 및 출력

```python
print(A + B + C)
```

* 세 정수를 더한 결과를 출력한다.
* 파이썬은 큰 정수도 자동으로 처리할 수 있어 별도의 예외 처리가 필요 없다.

---

### 메인 실행부

```python
if __name__ == '__main__':
    kid_jungmin()
```

* 해당 파일이 직접 실행될 때만 함수가 호출되도록 설정했다.
* 다른 파일에서 import 되었을 경우 자동 실행을 방지한다.

---

## 정리

* 여러 개의 값을 한 번에 입력받을 때 `map()`과 `split()`을 함께 사용하는 패턴은 매우 자주 등장한다.
* 파이썬은 정수 범위 제한이 없어 큰 수의 합도 안전하게 처리할 수 있다.
* 기본적인 입력 처리 연습에 적합한 문제이다.
