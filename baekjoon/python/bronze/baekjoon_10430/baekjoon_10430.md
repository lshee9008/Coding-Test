# 풀이 과정
# [백준] 10430번 나머지 (Python)
## 문제 설명

세 정수 `A`, `B`, `C`가 주어졌을 때,
아래 네 가지 연산 결과를 차례대로 출력하는 문제이다.

1. `(A + B) % C`
2. `((A % C) + (B % C)) % C`
3. `(A × B) % C`
4. `((A % C) × (B % C)) % C`

이 문제는 나머지 연산의 성질을 이해하고 있는지를 확인한다.

---

## 입력

* 한 줄에 세 정수 `A`, `B`, `C`가 공백으로 구분되어 주어진다.

## 출력

* 문제에서 제시한 네 가지 식의 결과를 각각 한 줄에 하나씩 출력한다.

---

## 풀이 접근

이 문제의 핵심은 **나머지 연산의 분배 법칙**이다.

다음 성질이 항상 성립한다.

* `(A + B) % C = ((A % C) + (B % C)) % C`
* `(A × B) % C = ((A % C) × (B % C)) % C`

문제에서는 이 성질이 실제 계산 결과에서도 동일함을 확인하도록
네 가지 식을 모두 출력하도록 요구한다.

풀이 흐름은 다음과 같다.

1. `input().split()`으로 세 값을 입력받는다.
2. `map(int, ...)`을 사용해 정수형으로 변환한다.
3. 문제에서 요구한 네 가지 연산을 순서대로 계산한다.
4. 각 결과를 한 줄씩 출력한다.

---

## 코드

```python
def the_rest():
    A, B, C = map(int, input().split())
    print((A + B) % C)
    print(((A % C) + (B % C)) % C)
    print((A * B) % C)
    print(((A % C) * (B % C)) % C)

if __name__ == '__main__':
    the_rest()
```

---

## 코드 분석

### 입력 처리 및 형 변환

```python
A, B, C = map(int, input().split())
```

* `input()`으로 한 줄 입력을 받는다.
* `split()`으로 공백 기준 분리한다.
* `map(int, ...)`을 통해 세 값을 모두 정수형으로 변환한다.
* 변환된 값이 각각 `A`, `B`, `C`에 할당된다.

---

### 덧셈 나머지 연산

```python
print((A + B) % C)
print(((A % C) + (B % C)) % C)
```

* 첫 번째 식은 두 수를 더한 뒤 나머지를 구한다.
* 두 번째 식은 각 수의 나머지를 먼저 구한 뒤 더하고 다시 나머지를 구한다.
* 두 결과는 항상 동일하다.

---

### 곱셈 나머지 연산

```python
print((A * B) % C)
print(((A % C) * (B % C)) % C)
```

* 곱셈에서도 동일한 나머지 연산 성질이 성립한다.
* 큰 수 연산에서 오버플로를 방지하기 위해 이 방식이 자주 사용된다.

---

### 메인 실행부

```python
if __name__ == '__main__':
    the_rest()
```

* 해당 파일이 직접 실행될 때만 함수가 호출된다.
* 다른 파일에서 import 되었을 경우 자동 실행을 방지한다.

---

## 정리

* 나머지 연산은 덧셈과 곱셈에 대해 분배 법칙이 성립한다.
* 이 성질은 알고리즘 문제에서 매우 자주 활용된다.
* 특히 큰 수 연산이나 암호학 관련 문제의 기초 개념이다.
