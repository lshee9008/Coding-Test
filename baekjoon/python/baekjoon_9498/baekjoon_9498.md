# [백준/Python] 9498번: 시험 성적

### 문제 링크
[백준 9498번: 시험 성적](https://www.acmicpc.net/problem/9498)

### 문제 설명
시험 점수를 입력받아 구간에 따라 학점을 출력하는 문제입니다.
- 90 ~ 100점: A
- 80 ~ 89점: B
- 70 ~ 79점: C
- 60 ~ 69점: D
- 나머지 점수: F

### 제출 코드

```python
def compare_grade():
    grade = int(input())
    
    if (grade <= 100 and grade >= 90):
        print("A")
    elif (grade <= 89 and grade >= 80):
        print("B")
    elif (grade <= 79 and grade >= 70):
        print("C")
    elif (grade <= 69 and grade >= 60):
        print("D")
    else:
        print("F")

if __name__ == '__main__':
    compare_grade()
```

### 코드 분석

작성한 코드는 파이썬의 기본 조건문인 `if-elif-else` 구조와 논리 연산자를 정확히 이해하고 활용한 좋은 예시입니다. 코드를 세 부분으로 나누어 분석해 봅니다.

**1. 입력 및 데이터 타입 변환**
```python
grade = int(input())
```
`input()` 함수를 통해 사용자의 입력을 받습니다. 파이썬에서 `input()`으로 받은 데이터는 기본적으로 문자열(str) 타입을 가지기 때문에, 점수의 크고 작음을 비교하기 위해 `int()`를 사용하여 정수형으로 변환(Casting)해 주었습니다.

**2. 조건 분기와 논리 연산자**
```python
if (grade <= 100 and grade >= 90):
    print("A")
```
논리 연산자 `and`를 사용하여 두 가지 조건을 모두 만족할 때 코드가 실행되도록 구현했습니다. 이후 10점 단위로 `elif`문을 사용하여 B, C, D 학점의 조건을 검사하고, 마지막으로 60점 미만의 모든 점수는 `else`문으로 처리하여 F를 출력하도록 안전하게 구성했습니다. 구간을 명확하게 한정 지어주었기 때문에 논리적인 구멍이 없는 구조입니다.

**3. 메인 실행 블록**
```python
if __name__ == '__main__':
    compare_grade()
```
단순한 알고리즘 문제 풀이를 넘어, 실제 프로그램 개발 시 권장되는 모듈화 구조를 사용했습니다. 해당 스크립트가 직접 실행될 때만 `compare_grade()` 함수가 호출되도록 설계된 점이 돋보입니다.

### 💡 파이썬 최적화 팁 (사슬 비교)

현재 코드도 훌륭하지만, 파이썬에서는 수학의 부등식처럼 비교 연산자를 연속으로 사용하는 **사슬 비교(Chained Comparison)** 기능을 제공합니다. 이를 활용하면 `and` 연산자를 생략하고 코드를 더 직관적으로 다듬을 수 있습니다.

```python
# 기존 방식
if (grade <= 100 and grade >= 90):

# 파이썬 사슬 비교 적용
if 90 <= grade <= 100:
```

또한, `if-elif` 문은 위에서부터 순차적으로 조건을 검사하여 내려오므로 아래와 같이 상한선 조건을 생략하여 작성하는 것도 가능합니다.

```python
if grade >= 90:
    print("A")
elif grade >= 80: # 이미 90점 이상은 위에서 처리되었으므로 자동으로 80~89점 구간이 됨
    print("B")
```
