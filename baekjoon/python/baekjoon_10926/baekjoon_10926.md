# 문제 출처
> https://www.acmicpc.net/problem/10926

# 풀이 과정
## 문제 설명

준하가 가입하려고 하는 사이트에 이미 존재하는 아이디가 주어진다.
아이디 뒤에 문자열 `??!`를 붙여서 출력하는 문제이다.

---

## 입력

* 첫째 줄에 준하가 가입하려고 하는 아이디가 주어진다.
* 아이디는 알파벳 소문자로만 이루어져 있다.

## 출력

* 입력으로 주어진 아이디 뒤에 `??!`를 붙여서 출력한다.

---

## 풀이 접근

입력으로 주어지는 아이디는 문자열이므로
추가적인 형 변환 없이 문자열 결합을 사용하면 된다.

문제 해결 과정은 다음과 같다.

1. `input()`으로 아이디를 입력받는다.
2. 입력받은 문자열 뒤에 `??!`를 덧붙인다.
3. 결과 문자열을 출력한다.

출력 로직은 함수로 분리하고,
`__main__` 구문을 사용해 실행 흐름을 명확히 했다.

---

## 코드

```python
def print_add_letter():
    name = input()
    name = str(name)
    print(name + '??!')

if __name__ == '__main__':
    print_add_letter()
```

---

## 코드 분석

### 입력 처리

```python
name = input()
```

* `input()`은 한 줄을 문자열로 입력받는다.
* 문제에서 아이디는 문자열이므로 그대로 사용 가능하다.

---

### 문자열 처리

```python
name = str(name)
```

* `input()`의 반환값은 이미 문자열이기 때문에 필수적인 코드는 아니다.
* 다만, 문자열임을 명확히 하기 위해 작성된 코드로 볼 수 있다.

---

### 문자열 결합 및 출력

```python
print(name + '??!')
```

* 문자열 `+` 연산자를 사용해 입력값 뒤에 `??!`를 붙인다.
* 문제에서 요구한 형식 그대로 출력한다.

---

### 메인 실행부

```python
if __name__ == '__main__':
    print_add_letter()
```

* 해당 파일이 직접 실행될 때만 함수가 호출되도록 설정했다.
* 다른 파일에서 import 되었을 때 자동 실행되는 것을 방지한다.

---

## 정리

* 문자열 문제에서는 형 변환보다 문자열 연산에 익숙해지는 것이 중요하다.
* `input()`은 기본적으로 문자열을 반환한다는 점을 기억하면 불필요한 변환을 줄일 수 있다.
* 간단한 출력 문제이지만, 코드 구조를 정리하는 연습에 적합하다.
