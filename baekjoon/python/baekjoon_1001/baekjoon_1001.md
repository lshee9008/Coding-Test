# 문제 출처
> https://www.acmicpc.net/problem/1001

# 풀이 과정
## 문제 설명

두 정수 `A`와 `B`가 주어졌을 때,
`A - B`의 결과를 출력하는 문제이다.

---

## 입력

* 한 줄에 두 정수 `A`, `B`가 공백으로 구분되어 주어진다.

## 출력

* `A - B`의 결과를 출력한다.

---

## 풀이 접근

입력으로 주어지는 값은 문자열 형태이기 때문에
정수 연산을 위해 형 변환이 필요하다.

이 문제에서는 `map()` 함수를 사용하여
입력과 동시에 정수형으로 변환하는 방식을 사용하였다.

전체 흐름은 다음과 같다.

1. `input().split()`으로 입력값을 공백 기준으로 분리한다.
2. `map(int, ...)`을 사용해 각 값을 정수로 변환한다.
3. 두 정수의 차를 계산하여 출력한다.
4. 실행 진입점을 명확히 하기 위해 `__main__` 구문을 사용한다.

---

## 코드

```python
def print_A_minus_B():
    A, B = map(int, input().split())
    print(A - B)

if __name__ == "__main__":
    print_A_minus_B()
```

---

## 코드 분석

### 함수 정의

```python
def print_A_minus_B():
```

* 두 정수를 입력받아 뺄셈 결과를 출력하는 함수이다.
* 연산 로직을 함수로 분리하여 코드의 역할을 명확히 했다.

---

### 입력 및 형 변환

```python
A, B = map(int, input().split())
```

* `input()`으로 한 줄을 입력받는다.
* `split()`으로 공백을 기준으로 문자열을 분리한다.
* `map(int, ...)`을 통해 각 값을 정수형으로 변환한다.
* 변환된 정수 값이 각각 `A`, `B`에 할당된다.

---

### 연산 및 출력

```python
print(A - B)
```

* 정수형으로 변환된 `A`와 `B`의 차를 계산하여 출력한다.

---

### 메인 실행부

```python
if __name__ == "__main__":
    print_A_minus_B()
```

* 해당 파일이 직접 실행될 경우에만 함수가 호출되도록 설정했다.
* 모듈로 import 되었을 때 불필요한 실행을 방지한다.

---

## 정리

* `map()`을 사용하면 입력과 형 변환을 한 줄로 처리할 수 있다.
* 간단한 문제라도 함수와 실행 구문을 분리하면 코드 구조가 깔끔해진다.
* 이후 여러 입력을 다루는 문제에서 동일한 패턴을 활용할 수 있다.
