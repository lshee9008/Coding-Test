# 문제 출처
> https://www.acmicpc.net/problem/1000

# 풀이 과정
## 문제 설명

두 정수 `A`와 `B`가 주어졌을 때,
두 수의 합을 출력하는 문제이다.

---

## 입력

* 한 줄에 정수 `A`, `B`가 공백으로 구분되어 주어진다.

## 출력

* `A + B`의 값을 출력한다.

---

## 풀이 아이디어

입력은 문자열 형태로 들어오기 때문에 다음 과정을 거쳐야 한다.

1. `input()`으로 한 줄 입력을 받는다.
2. `split()`을 사용해 공백 기준으로 두 값을 분리한다.
3. 문자열을 정수형으로 변환한 뒤 덧셈을 수행한다.
4. 결과를 출력한다.

간단한 문제이지만, 함수로 분리하고 `__main__` 구문을 사용해 코드 구조를 명확히 했다.

---

## 코드

```python
def print_A_add_b():
    A, B = input().split()
    print(int(A) + int(B))

if __name__ == '__main__':
    print_A_add_b()
```

---

## 코드 설명

### 입력 처리

```python
A, B = input().split()
```

* `input()`은 한 줄을 문자열로 입력받는다.
* `split()`을 사용해 공백을 기준으로 두 값을 나눈다.
* 나뉜 값은 문자열이므로 바로 계산할 수 없다.

---

### 형 변환 및 출력

```python
print(int(A) + int(B))
```

* 문자열 상태로 더하면 숫자 덧셈이 아니라 문자열 결합이 된다.
* `int()`로 정수형으로 변환한 후 덧셈을 수행한다.

---

### 메인 실행부

```python
if __name__ == '__main__':
    print_A_add_b()
```

* 해당 파일이 직접 실행될 때만 함수가 호출되도록 설정했다.
* 모듈로 import 될 경우 실행되지 않게 하기 위한 파이썬의 기본 구조이다.

---

## 정리
* `input()`, `split()`, `int()`의 사용법을 익히기에 적합하다.
* 간단한 문제라도 코드 구조를 신경 쓰면 가독성과 재사용성이 좋아진다.
