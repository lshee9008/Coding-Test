# [백준] 1330번 두 수 비교하기 (Python)

# 문제 사이트
> [https://www.acmicpc.net/problem/1330](https://www.acmicpc.net/problem/1330)

---

## 문제 설명

두 정수 `A`와 `B`가 주어졌을 때,
`A`와 `B`의 대소 관계를 비교하여 아래 세 가지 중 하나를 출력하는 문제이다.

* `A > B`인 경우 `>`
* `A < B`인 경우 `<`
* `A == B`인 경우 `==`

---

## 입력

* 한 줄에 두 정수 `A`, `B`가 공백으로 구분되어 주어진다.

## 출력

* 문제에서 요구한 비교 결과를 출력한다.

---

## 풀이 접근

이 문제는 파이썬의 조건문을 사용해
두 값의 대소 관계를 판단하면 된다.

풀이 흐름은 다음과 같다.

1. `input().split()`으로 두 값을 입력받는다.
2. `map(int, ...)`을 사용해 정수형으로 변환한다.
3. `if`, `elif` 조건문을 사용해 값을 비교한다.
4. 조건에 맞는 문자열을 출력한다.

---

## 코드

```python
def compare():
    A, B = map(int, input().split())
    if A > B:
        print('>')
    elif A < B:
        print('<')
    elif A == B:
        print('==')

if __name__ == '__main__':
    compare()
```

---

## 코드 분석

### 입력 처리 및 형 변환

```python
A, B = map(int, input().split())
```

* `input()`으로 한 줄 입력을 받는다.
* `split()`으로 공백 기준 분리한다.
* `map(int, ...)`으로 두 값을 정수형으로 변환한다.

---

### 조건문 비교

```python
if A > B:
    print('>')
elif A < B:
    print('<')
elif A == B:
    print('==')
```

* `if`문으로 `A`가 `B`보다 큰 경우를 먼저 검사한다.
* `elif`로 작은 경우를 검사한다.
* 앞의 조건에 모두 해당하지 않으면 두 값은 같으므로 `==`를 출력한다.
* 모든 경우가 조건문으로 명확하게 분리되어 있다.

---

### 메인 실행부

```python
if __name__ == '__main__':
    compare()
```

* 해당 파일이 직접 실행될 때만 함수가 호출된다.
* 다른 파일에서 import 되었을 경우 자동 실행을 방지한다.

---

## 정리

* 조건문의 기본 구조(`if`, `elif`)를 익히기에 적합한 문제이다.
* 비교 연산자(`>`, `<`, `==`)의 사용법을 확인할 수 있다.
* 출력 형식이 정확해야 하므로 문자열 출력에도 주의해야 한다.
